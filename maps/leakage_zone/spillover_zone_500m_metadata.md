## Method

The leakage zone was generated by applying a 500 m outward buffer to the
Jaunmārupe project boundary (source: `04_GIS/01_polygons/Jaunmarupe_Project_Latvia.gpkg`)
using `geopandas.buffer()` in EPSG:3035 (equal-area projection) to ensure
accurate distance measurement. The resulting polygon represents the area
surrounding the project site that falls within the leakage monitoring zone
as defined in the Wallacea Trust biodiversity credit application framework.

The script that generated this file is `05_analyses/leakage_zone_map.py`.
